Build a React TypeScript dashboard application using Vite for a van control system called MTVC Van Control System by Mobile Tyre Van City. This runs on a Raspberry Pi 5 with a 7" touchscreen at 800×480px. Use Tailwind CSS for styling.
Tech stack:

React 18 + TypeScript
Vite
Tailwind CSS
React Router for page navigation

The app has two separate views:
/ — Main panel (800×480px) for the power cabinet touchscreen
/cab — Cab display (1280×400px) for a bar screen mounted at the top of the windscreen

GLOBAL COMPONENTS
Create these reusable components in /src/components/:
ArcGauge.tsx — SVG arc gauge, 270° sweep, rounded linecaps, takes value, max, size, color, children props. Shows a track ring and a coloured fill ring. Children render centred inside.
HBar.tsx — Horizontal progress bar, takes pct (0-100) and color props. Dark track with coloured fill and subtle glow box-shadow.
StatRow.tsx — A row with a dim uppercase label on the left and a monospace value on the right. Optional left accent border. Dark surface2 background.
Card.tsx — Dark surface card with 1px border, 2px coloured top border (accent prop), 4px border radius, subtle box-shadow. Transitions on background and border for theme switching.
Toggle.tsx — A pill-shaped on/off toggle switch. Takes on (boolean), onToggle, color props. Smooth left/right thumb animation.
ConnDot.tsx — A small coloured dot with glow box-shadow. Green when connected, red when not. Gentle pulse animation.

Nav bar:
6 equal tabs: Inverter ⚡ · Battery 🔋 · Air 💨 · Fans ⚙ · Lights 💡 · Status 📊

6 content pages (each absolutely positioned, fills content area, padding 12px, flex column gap 10px, overflow-y auto):
Page 1 — Inverter:

Middle: Card "CELL VOLTAGES — 16S" — 8-column grid of 16 cell tiles. Each tile: C1 label, voltage in monospace coloured green/amber/red, tiny 3px fill bar at bottom

"LIGHTING" label + active count chip. 4×2 grid of compact zone pill toggles (tap to toggle, amber when on, red for Emergency). "CABINET FANS" label below. 3 fan status tiles (Cabinet, Exhaust, Aux) with spinning ⚙ when active.
Column 4 (flex-1, right side nearest driver) — Emergency:
Centred vertically. "EMERGENCY" or "ACTIVE" status label above button. Large circular button (160×160px): deep red radial gradient, 🚨 emoji, "EMERG" text. When active: red glow box-shadow, pulse animation, whole column gets subtle red radial background glow. "TAP TO ACTIVATE / DEACTIVATE" label below. "⚠ LIGHTS ON" pulsing chip appears below when active.

ADDITIONAL REQUIREMENTS
Theme switching: implement via React context (ThemeContext). Provide dark and light token sets. All components read colours from context. Transition all colour properties with transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1).
Environment config: create .env.development with VITE_API_URL=http://localhost:8000 and .env.production with VITE_API_URL=http://vanctrl.local:8000.
